# WITI Viral Vision - AI Video Generation Engine for Wix/Shopify

[![AI](https://img.shields.io/badge/AI-Replicate-black?style=for-the-badge)](https://replicate.com)

**Witi Viral Vision** is a high-end full-stack AI-powered video automation engine designed for e-commerce platforms like Wix and Shopify. It uses cutting-edge GenAI models to transform product photos into high-engagement viral videos.

## 🚀 Key Features
- **AI-Powered Generation**: Integration with Replicate (Minimax Video-01 models).
- **Credit-Based Monetization**: Full logic for user credits management.
- **Seamless SaaS Integration**: Built-in support for Wix SDK and OAuth flows.
- **Robust Backend**: Node.js/Express architecture with MongoDB for persistent storage.

## 🛠️ Tech Stack
- Backend: Node.js, Express, MongoDB
- AI: Replicate API (GenAI Video)
- Integration: Wix App SDK / OAuth 2.0
